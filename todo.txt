Export:
1.export a=1 b=2 c=3 - добавляться должны все 3 переменные, записанные таким образом
2.export _abc=10 - должно добавлять новую переменную окружения (сейчас пишет not a valid identifier)

Unset:
1. unset a1=10 - результат должен быть "bash: unset: `a1=10': not a valid identifier" (сейчас не выдает ничего)
2. unset a b c - удаляться должны все 3 переменные, записанные таким образом
3. unset HOME - чистит все переменные окружения вместо HOME - только в CLion. Magic

Ctrl + D:
1. Должен не делать ничего, когда в строку введены символы
2. Должен выходить с надписью "exit" в остальных случаях

Pointer being freed was not allocated:
1. Вылетает сега если строка начинается с ;
